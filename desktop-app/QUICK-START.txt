  🚀 روش سریع (3 دقیقه) | Quick Method (3 minutes)
═══════════════════════════════════════════════════════════════════

1. Node.js را نصب کنید (اگر ندارید):
   https://nodejs.org/
   (دکمه سبز بزرگ را بزنید و نصب کنید)

2. فایل build.bat را دابل کلیک کنید

3. صبر کنید تا ساخت تمام شود

4. فایل EXE در پوشه dist آماده است!


═══════════════════════════════════════════════════════════════════
  📁 فایل‌های خروجی | Output Files
═══════════════════════════════════════════════════════════════════

پس از ساخت، این فایل‌ها ایجاد می‌شوند:

dist/
├── Engineering Calculator Setup-x64.exe    ← نصب‌کننده 64 بیتی
├── Engineering Calculator Setup-ia32.exe   ← نصب‌کننده 32 بیتی  
├── Engineering Calculator-Portable.exe     ← نسخه پرتابل
└── win-unpacked/                           ← فایل‌های استخراج شده
    └── Engineering Calculator.exe


═══════════════════════════════════════════════════════════════════
  ⚠️ عیب‌یابی | Troubleshooting
═══════════════════════════════════════════════════════════════════

❓ اگر خطای "'npm' is not recognized" دیدید:
   → Node.js نصب نیست. از nodejs.org دانلود کنید.

❓ اگر خطای دسترسی دیدید:
   → Command Prompt را با "Run as Administrator" اجرا کنید.

❓ اگر فایل index.html پیدا نشد:
   → ابتدا پروژه اصلی را با "npm run build" بسازید.


═══════════════════════════════════════════════════════════════════
  📋 سیستم مورد نیاز | System Requirements
═══════════════════════════════════════════════════════════════════

• Windows 7/8/10/11
• 100MB فضای دیسک
• 2GB RAM
• برای ساخت: Node.js 18+
